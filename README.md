# Sanika Dabade — Portfolio

Personal portfolio site for Sanika Dabade, a Java Full Stack Developer (Spring Boot + React).

Live: https://sanika-dabade-portfolio.vercel.app

## Stack

- React 19 + Vite
- Tailwind CSS v4
- Framer Motion (scroll animations)
- Lucide / react-icons

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
